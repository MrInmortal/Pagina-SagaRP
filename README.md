# 🌌 ABOUT THE PROJECT

> **SAGA RP** is a modern cinematic website built for a FiveM Roleplay community focused on immersion, aesthetics, and premium user experience.

This platform combines:
- 💚 Futuristic Neon UI
- 🏠 Interactive Housing System
- 📜 Dynamic Rules Pages
- 🎮 Controls & Commands System
- 🌌 Aurora Visual Effects
- ⚡ Smooth Animations
- 📱 Fully Responsive Design
